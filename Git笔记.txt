Git

什么是Git？
    最先进的版本控制系统
            分布式（参与者下载和上传资料至中心服务器，可下载编辑完成后上传）
            集中式（参与者下载和上传资料至中心服务器，要随时下载修改随时上传，必须全天运行）

   版本：HEAD   
        HEAD^:上一个版本

        HEAD~100:前第100个版本
        版本号：git log命令执行
    前进版本  git reset --hard （git reflog中的版本号）
